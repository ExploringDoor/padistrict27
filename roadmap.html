Phase 1 · Real Launch

⏱ NEXT · 6 WEEKS Pending Jeff's greenlight · soft-launch July 15

The 8 backend deliverables that turn the demo into a working district platform. After Phase 1, one league + one tournament are live end-to-end on production. Demo data is replaced with real D27 data.

Foundation

Authentication & roles

Firebase Auth · magic-link login · Admin/President/Captain/Player role detection. Preview →

Foundation

Database migration

Firestore schemas: leagues, teams, rosters, games, pitch counts, comms. All hardcoded HTML data → JSON.

Integration

Sports Connect bridge

Nightly roster sync from SC (LL-mandated registration). Preview →

Integration

GameChanger PDF import

Captain drag-drops post-game PDF · system parses + publishes box score. Preview →

Foundation

Stripe Connect

Registration · donations · sponsor invoices · store · auto-split. Preview →

Foundation

Email + SMS broadcasting

SendGrid + Twilio · rainouts, RSVP reminders, cert lapse alerts.

Integration

JDP / Safesport / Concussion API

Auto-mark coaches cleared/expired (currently manual). 30-day pre-lapse alerts.

Foundation

ICS calendar feeds

Subscribe links → live Google/Apple/Outlook updates with rainout sync.

Phase 2 · Mobile + Live Game

📱 MONTHS 2–4 After Phase 1 stable

Take the platform mobile and turn live games into real-time experiences. After Phase 2, parents can score from a phone offline, all 12 leagues are running, and parents at home can watch the bracket update pitch-by-pitch.

Mobile

PWA installable

Add to home screen on iOS/Android. Service worker for offline mode.

Mobile

Push notifications

Firebase Cloud Messaging. Game alerts, score updates, rainouts in 30 seconds.

Mobile

Captain offline mode

Score a remote-field game with no wifi. Auto-sync when back online.

Live

Real-time scoring

WebSocket. LIVE badges update every pitch. Preview →

Live

Embeddable scoreboards

Widgets for league sites that want bracket data on their own pages.

Safety

Concussion incident reporting

PA Safety in Youth Sports Act-compliant. Required by state law.

Safety

Heat-index + lightning alerts

Auto-suspend play threshold notifications. Insurance-mandated.

Safety

EAP per field · QR code

Emergency Action Plan accessible from dugout via QR code.

Safety

Injury log

Track every field incident · insurance + LL HQ reporting.

Phase 3 · Player Development & Depth

🏆 MONTHS 4–8 First full season on production

Compete head-on with GameChanger and TeamSnap on the things they do — plus the things they don't. After Phase 3, D27 doesn't just track games — it actively makes players better.

Player Dev

Skill assessment tools

Exit velo, pitch velo, sprint times. Pairs with HitTrax/Rapsodo if leagues have them.

Player Dev

Player Dev

Video review

Player Dev

Live

AI game recaps

Live

Audio broadcast streams

Live

Video streaming

GameChanger Live or YouTube Live integration on game pages.

Comms

Newsletter builder

Monthly D27 digest, Mailchimp-style. Auto-pulls game results & scholarship news.

Comms

Comms

Coaches' clinic archive

Video library of past D27 clinics. Replay + transcript.

Revenue

Sponsor analytics dashboard

Show Wawa/MLH their ROI. Justifies $5K renewals.

Revenue

Revenue

Auction + raffle tools

Revenue

Phase 4 · Multi-District Platform

🌐 YEAR 2+ If D27 succeeds, this becomes a business

Once D27 is running smoothly, the question becomes: can other districts use it too? Phase 4 is white-labeling the platform, building native apps, and integrating with the broader baseball ecosystem. This is where it stops being a pet project and starts being a real product.

Platform

White-label districts

Sell the stack to D24, D31, etc. ~$2–4K/season × 100 districts in PA/NJ/MD = real recurring.

Platform

60-min new district onboarding

Charter ID, league import, theme — automated.

Platform

Cross-district tournaments

Section 8 host pulls bracket data from all participating districts.

Mobile

iOS app (App Store)

TestFlight beta → public release. Wraps PWA initially.

Mobile

Android app (Play Store)

Same as iOS — wraps PWA, then native for performance.

Mobile

Apple Watch / WearOS

Live game alerts on wrist. Score subs, rainouts.

Integration

Daktronics scoreboards

Push from D27 to physical scoreboards on field.

Integration

HitTrax / Rapsodo

Pull skill assessment data into player profiles automatically.

Integration

PerfectGame profiles

Junior League players' showcase data linked to D27 profiles.

Integration

MaxPreps continuity

Junior/Senior League → high school transition tracking.

Analytics

At-risk player flagging

Kids missing games or with declining stats — flag to coach for outreach.

Analytics

Quality Charter auto-scorecard

Track all 23 LL HQ metrics. Auto-submit annual report.

Revenue

Estate + planned giving

Revenue

Premium captain features
