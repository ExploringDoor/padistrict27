# League / Team logos

Drop each league's logo here, named with the slug below (lowercase).
**PNG with a transparent background** is ideal; SVG also works. Square-ish crops
look best (they render in a circle/small box). One file per league — it auto-maps
to every team variant (e.g. `coventry.png` covers "Coventry (Keeler)",
"Coventry (Vermillion)", etc.).
