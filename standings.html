How standings power tournament seeding

When the regular season closes June 1, top finishers in each division qualify for D27's bracket tournaments (Bob Jones, Pappy Patton, etc.). Win-percent is primary; head-to-head and run differential break ties. Live game scoring updates these tables in real time — no more hand-tabulated standings or "the website hasn't been updated since April" emails. League presidents see only their teams' records via single-league filter; D27 admin sees the whole district.
